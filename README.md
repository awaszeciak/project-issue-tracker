# Project Issue Tracker

Backend REST API for managing tasks and issues in projects.

## Tech stack

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Swagger


## Features

- Create task
- Get all tasks
- Get task by ID
- Update task
- Delete task
- Filter tasks by status
- Filter tasks by priority
- Request validation

## API documentation
After running the application, Swagger UI is available at:
http://localhost:8081/swagger-ui/index.html

```bash
./mvnw spring-boot:run
